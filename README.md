# Employee Manager 

## About 
One of my dotnet projects. </br>
The program's primary purpose is to process employee lists and personal data, including registering new employees, 
calculating their expected salaries, uploading their photos, sorting the overall employee list, 
displaying the information in a table, displaying employee photos, and deleting employee data. 
The employee list is stored as a text file. </br>
Learning Windows Forms.
